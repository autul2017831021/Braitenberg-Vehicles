# Braitenberg-Vehicles
This repo contains the implementation of Valentino Braitenberg Vehicles.  The implemented vehicles explore the behavior-based robotics, cybernetics, and artificial life described in the book.

# How to Run?
Install python3.

<h2>Vehicle 6</h2>
    1. Go to the directory <b>Vehicle-6</b> in your terminal
    2. Make a virtual environment(the python location in your device) : 
       <b>python3.12 -m venv .venv</b>
    2. Type these commands :
       <b>source .venv/bin/activate</b>
       <b>python -m pip install pygame</b>
    3. Get out of the Virtual Environment.
    4. Now write this command :
       <b>./main.sh</b>
